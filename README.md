### ✨ Sobre o Projeto
Este é um projeto de **Carta de Pedido de Namoro** interativa. Ele foi construído com uma estética retrô/pixel art, focado em criar uma experiência lúdica e fofa. O projeto conta com:

* 💌 **Envelope Animado:** Uma introdução charmosa que convida ao clique.
* 🏃‍♂️ **Botão Fujão:** Uma brincadeira clássica onde o "Não" foge do cursor.
* 📈 **Persistência:** O botão "Sim" cresce a cada tentativa de fuga do "Não".
* 🎁 **Surpresa Final:** Uma camada extra de interação com artes personalizadas em pixel art.

---

### 🛠️ Tecnologias Utilizadas
Para dar vida a essa ideia, utilizei o "trio de ouro" do front-end:

* **HTML5** (Estrutura)
* **CSS3** (Estilização, animações e layout)
* **JavaScript** (Lógica do botão fujão e transições de tela)
* **Google Fonts** (Fonte *Pixelify Sans*)
